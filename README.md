## 6. 서비스 실행 방법

레포지토리를 clone 받아야합니다!

```shell
$ git clone https://github.com/devRangers/wonderingPill.git
$ cd wonderingPill
```

각 파트별 실행이 필요합니다.

- **Front**

```shell
$ cd front
$ yarn install
$ yarn dev
```

- **Back**

@ParkSuJeong 에게 `*.env`를 요청해주세요!

```shell
$ cd back
$ npm install
$ npm run start:dev
```

- **AI**

```shell
$ cd ai
```
